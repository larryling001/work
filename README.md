# 说明
本系统使用js+html+css实现，整体采用子使用布局，页面中的内容实现了过渡动画 svg动画 音频播放 模拟页面切换功能
# 项目目录及说明
css 存放css相关内容
font 存放字体
js 存放js文件
static 存放图片音乐等静态资源
